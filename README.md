About Webbed Dreams

Webbed Dreams is a high-end web design agency specializing in immersive, glass-morphism aesthetic and futuristic UI/UX. We bridge the gap between abstract vision and functional reality.

This repository contains the source code for our portfolio website. It acts as a showcase of our technical capabilities, featuring advanced particle animations, React architecture, and responsive Tailwind styling.

The Vibe & Design System

Theme: "Dark Ether" (Deep Navy/Black backgrounds)

Accents: Electric Cyan (#06B6D4) & Neon Purple (#7C3AED)

Typography:

Headings: Space Grotesk (Futuristic, Clean)

Body: Inter (Readable, Modern)

Visuals: Glassmorphism, subtle gradients, and connectivity-themed animations.

Tech Stack

This project is built as a Single Page Application (SPA) using modern web technologies:

Core: React 18

Styling: Tailwind CSS (Utility-first framework)

Animation:

Framer Motion (UI transitions)

HTML5 Canvas (Custom particle background "The Web")

Icons: Lucide React

Fonts: Google Fonts

Getting Started

Prerequisites

You need a modern web browser to view this project. No Node.js environment is strictly required for the index.html version as it utilizes CDN imports for rapid prototyping and ease of sharing.

Installation & Deployment

Clone the repository:

git clone [https://github.com/yourusername/webbed-dreams.git](https://github.com/yourusername/webbed-dreams.git)


Run Locally:
Simply open the index.html file in your preferred web browser (Chrome, Firefox, Safari).

Deploy:
This project is optimized for static hosting.

GitHub Pages: Go to Settings -> Pages -> Select 'main' branch -> Save.

Vercel/Netlify: Drag and drop the folder containing index.html.
